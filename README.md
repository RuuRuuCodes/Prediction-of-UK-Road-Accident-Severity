# Prediction-of-UK-Road-Accident-Severity
This work focuses on employing data mining techniques for uncovering patterns that can be utilized for predicting the severity of injuries in road accidents.  

This work utilizes a dataset that provides information regarding the severity of road accidents in the United Kingdom (UK). The dataset utilized is available at: https://www.kaggle.com/datasets/salmankhaliq22/road-traffic-collision-dataset. It contains records of road traffic collisions that occurred in the UK between 2005 and 2017.
